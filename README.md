# T3 Code Mobile

A standalone Expo client for T3 Code environments. It provides project and
thread browsing, thread chat, and Git commit/push/pull-request actions.

## Run

Use Node 24.13.1 or newer within the Node 24 release line.

```sh
pnpm install
pnpm dev
```

Open the Environments screen and paste a T3 Code pairing URL. The bearer
session is stored in the platform secure store.
